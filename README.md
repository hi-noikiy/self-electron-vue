# trade_bot

> cryptocurrency exchange bot



#### Build Setup

``` bash
使用yarn安装打包  不然会出错


# install dependencies
yarn       #ccxt模块需要链接外网

# serve with hot reload at localhost:9080
yarn  dev

# build electron application for production
yarn  build

# run unit & end-to-end tests
yarn test




```
